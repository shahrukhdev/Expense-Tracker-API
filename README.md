Expense Tracker API

A RESTful Expense Tracker API built with Node.js, TypeScript, Express, and MongoDB. It provides endpoints for managing income, expenses, categories, recurring transactions, and budgets, enabling efficient personal finance tracking and integration with frontend applications.

Features
Manage Income & Expenses
Create and organize Categories
Handle Recurring Transactions
Set and monitor Budgets
RESTful API design
Built with TypeScript for type safety
Scalable and clean architecture

Tech Stack
Node.js
TypeScript
Express.js
MongoDB (Mongoose)
